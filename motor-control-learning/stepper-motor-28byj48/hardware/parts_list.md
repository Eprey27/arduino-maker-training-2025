## Especificaciones Técnicas Detalladas

### Motor 28BYJ-48

- **Voltaje:** 5V DC
- **Pasos por revolución (modo full-step):** 2048
- **Pasos por revolución (modo half-step):** 4096
- **Reducción de engranajes:** 64:1 (aproximado, real ~63.68:1)
- **Torque:** ~34 N·cm (a 5V)
- **Velocidad máxima:** ~15 RPM
- **Consumo:** 120-240 mA
- **Peso:** ~30g
- **Dimensiones:** 28mm diámetro × 19mm altura

### Driver ULN2003

- **Chip:** ULN2003APG (Darlington Array)
- **Canales:** 7 (usamos 4 para motor)
- **Corriente por canal:** 500 mA máx
- **Voltaje entrada:** 5V TTL
- **Voltaje salida:** Hasta 50V
- **Incluye:** LEDs indicadores, conectores

### Sensor AS5600

- **Resolución:** 12 bits (4096 posiciones)
- **Precisión:** ±0.35° (typ)
- **Interfaz:** I2C (dirección 0x36)
- **Voltaje:** 3.3V - 5V
- **Consumo:** 6.5 mA típico
- **Velocidad I2C:** Hasta 1 MHz
- **Frecuencia actualización:** 145 Hz (típico)
- **Air gap:** 0.5mm - 3mm
- **Temperatura operación:** -40°C a +125°C

### Arduino UNO

- **Microcontrolador:** ATmega328P
- **Voltaje operación:** 5V
- **Entradas/Salidas digitales:** 14 (6 PWM)
- **Entradas analógicas:** 6 (10-bit ADC)
- **Memoria Flash:** 32 KB
- **SRAM:** 2 KB
- **EEPROM:** 1 KB
- **Clock:** 16 MHz
- **Corriente I/O:** 20 mA por pin (max 40 mA)

### Placas Compatibles (Alternativas a Arduino UNO)

| Placa | Compatible | Notas |
|-------|------------|-------|
| Arduino Nano | ✓ | Cambiar pines, más compacto |
| Arduino Mega | ✓ | Más recursos, mismo código |
| ESP32 | ✓ | Cambiar pines I2C (GPIO21/22) |
| ESP8266 | ✓ | Cambiar pines, menos memoria |
| STM32 Blue Pill | ⚠️ | Requiere adaptación de librerías |

## Checklist de Compra

Antes de ordenar, verificar:

- [ ] Versión correcta del motor (5V, no 12V)
- [ ] AS5600 incluye imán diametral
- [ ] Arduino tiene cable USB incluido
- [ ] Suficientes cables para todas las conexiones
- [ ] Breadboard si no quieres soldar
- [ ] Verificar voltajes compatibles (3.3V vs 5V)
- [ ] Revisar especificaciones del vendedor
- [ ] Leer reseñas de otros compradores
- [ ] Comparar precios entre proveedores
- [ ] Considerar tiempo de envío vs presupuesto
